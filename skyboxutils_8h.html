vtkSmartPointer< vtkOpenGLTexture > LoadCubemapTexture ( const std::vector< std::string > &  faceFilenames)

Loads a cubemap texture from six image files.

This function loads six images representing the faces of a cubemap and returns a VTK OpenGL texture that can be used in a skybox.

Parameters
faceFilenamesA vector of 6 file paths in the order: right, left, top, bottom, front, back.
Returns
A smart pointer to the loaded cubemap texture.

This function creates a VTK OpenGL cubemap texture using six image files. Each image is flipped on the Y-axis to match OpenGL's texture coordinate system.

Parameters
faceFilenamesA vector of 6 file paths for the cube faces, in this order: +X (right), -X (left), +Y (top), -Y (bottom), +Z (front), -Z (back).
Returns
A smart pointer to the resulting vtkOpenGLTexture.
